Brief

In the game 2 players move in four directions:
  • UP
  • DOWN
  • RIGHT
  • LEFT
In each turn player has to make one move. Each player leaves a trace of his movement. When the player hits the border of the board or the trace of one of the players (including his own trace) he loses. The aim of the game is to outlive the opponent.

Input

The input consist of several parts:
  • In the first line there are 2 integers N, nr denoting the size of the board and player number
  • In each the next 2 lines there are 2 integers r, c, denoting the position of the player number k (k = 0, 1). r - row number, c - column number.
  • In the next N lines there are N integers. That describes the board state. Their values mean:
    • -1 - the field is empty
    • i - the field is occupied by the trace of i-th player

Output

You should print one string ("UP", "DOWN", "RIGHT", "LEFT") - your move.
